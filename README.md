## step 1

You just need to customize **wpmMin** and **wpmMax**
```js
const wpmMin = 150;
const wpmMax = 160;
```
Reading Time is between [wpmMin, wpmMax], including **wpmMin** and **wpmMax**

open developer tool

2. F12
